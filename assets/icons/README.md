# Tool icons for the desktop UI

Place your PNG icons here with these exact names:

- **blender.png** – Blender tool
- **obsidian.png** – Obsidian tool  
- **krita.png** – Krita tool

Recommended size: at least 80×80 px (they are displayed at 80×80).  
If a file is missing, the app falls back to text labels.
